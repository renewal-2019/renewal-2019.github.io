# 密封件测绘 Wiki 静态页

这是一个可直接部署到 GitHub Pages 的静态站点。

当前页面内容已经按照思维导图整理为以下结构：

- 密封件测绘
  - 新环 / 槽测绘
    - 气缸环
      - 活塞环
      - 支撑环
        - 普通环形
        - 特殊环形
    - 填料环
      - 普通环形
      - 特殊环形
    - 填料盒
      - 法兰
      - 接头
  - 失效测绘
    - 气缸环
    - 填料环
    - 填料盒

## 文件说明

- `index.html`：主页内容
- `styles.css`：样式文件
- `script.js`：页面交互

## 部署方式

1. 新建 GitHub 仓库。
2. 把这 3 个文件上传到仓库根目录。
3. 在 GitHub 仓库设置中开启 `Pages`。
4. Source 选择当前分支的 `/ (root)`。
5. 保存后等待 GitHub 分配访问地址。

## 后续修改建议

- 直接编辑 `index.html` 里的各个章节文字即可。
- 如果你后面要增加更多页面，可以继续拆分成 `materials.html`、`process.html` 之类的独立页面。
- 如果需要放实物照片，可以在仓库中新建 `images/` 目录，然后在 `index.html` 里引用。
